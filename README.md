# Basic-Network-Traffic-Analysis-Lab
A very basic traffic analysis lab wehere I demonstrate my ability to install wireshark, its dependancies, then view network traffic


Summary:

This project demonstrates basic network traffic capture and analysis using Wireshark in a virtualized Ubuntu environment. The goal is to observe and understand how computers communicate over a network by analyzing different protocols.

Key activities include:

Setting up a virtual lab using VMware and Ubuntu.

Capturing live network traffic with Wireshark.

Filtering and analyzing DNS queries, TCP handshakes, and ICMP ping packets.

Observing source and destination IPs, ports, and protocol behavior.

Documenting network activity to understand how devices resolve domain names and establish reliable connections.

Tools Used: VMware Workstation, Ubuntu, Wireshark
Skills Demonstrated: Network analysis, protocol inspection, packet capture, basic cybersecurity concepts
